Enables you how many times a files (in one of 5 mirrors server) had been downloaded.

== Description ==

This plugin shows you , hits of your download . You can add 5 mirrors server for your files. 

Languages:

* Persian [Reza-Sh](http://rezaonline.net/blog/ "رضا شیخله")
* English [Reza-Sh](http://rezaonline.net/blog/)

Send email to me for your language Translations pack: reza19sh[a]gmail[dot].com
(In plugins package have .pot file)

Features:

* Counts your download files
* Easy installation and use
* Settings for best viwed
* Have a simple download chart
* You can add 5 mirrors link for your file

Usage:

1. upload your file in max 5 servers like 4shared , ...
2. go to add files in plugins menu and fill it ,
3. In your post/page for viewed list of mirrors use [dlmc id="1"] 
4. In your post/page for viewed hits of downloaded files use [dlmc countid="1"]

== Installation ==

1. Upload the directory `/wp-download-mirror-counter/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
